# Twostar Pharmaceutical Trading - Landing Page

A modern, responsive React website for Twostar Pharmaceutical Trading (TSP), a Filipino-owned pharmaceutical company..

## Tech Stack

- **React 18** with **TypeScript**
- **Vite** for build tooling
- **Tailwind CSS** for styling
- **React Router** for navigation
- **lucide-react** for icons

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

## Project Structure

```
src/
  components/     # Reusable UI components
  pages/          # Page components
  data/           # Data models and hardcoded content
  App.tsx         # Main app component with routing
  main.tsx        # Entry point
  index.css       # Global styles
```

## Pages

- `/` - Home page with hero section, services, and client preview
- `/about` - Company overview, mission/vision, locations, and full client list
- `/products` - Product catalog with search and category filters
- `/careers` - Job openings and company benefits
- `/contact` - Contact form and location information

## Features

- Responsive design (desktop-first, mobile-friendly)
- Gradient hero sections
- Floating stat cards and chips
- Product catalog with filtering
- Contact form with inquiry type selection
- Image placeholders (ready for image replacement)

## Image Placeholders

All images use placeholder components. Replace the following image paths with actual images:

- `/images/hero-main.jpg` - Hero section main image
- `/images/warehouse.jpg` - Warehouse location image
- `/images/product-placeholder.jpg` - Product catalog images
- `/images/careers.jpg` - Careers page image
- `/images/map.jpg` - Contact page map
